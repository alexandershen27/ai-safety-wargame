# Feature specs

One file per planned feature. **Read these before opening a PR** for any of
them — there are real design decisions baked in that aren't visible from the
schema alone.

| Spec | Priority | What it adds |
|---|---|---|
| [forced-response.md](forced-response.md) | v0.2 | Tag a role with your action → they get a forced response slot next turn |
| [world-state-and-deltas.md](world-state-and-deltas.md) | v0.2 | Typed delta pills + always-visible metric strip |
| [world-templates.md](world-templates.md) | v0.3 | Presets ("AI 2027", etc.) that seed roles + metrics |
| [gm-cockpit.md](gm-cockpit.md) | v0.3 | Drag-reorder, edit-live, NPC actions, kick player |
| [notebook-and-sources.md](notebook-and-sources.md) | v0.3 | Multi-file markdown + sources list per player |
| [branching-and-rollback.md](branching-and-rollback.md) | v0.4 | Rewind to past turn; alternate timelines |
| [secrets-and-visibility.md](secrets-and-visibility.md) | v0.4 | Per-action visibility (only some roles see) |
| [multi-player-per-role.md](multi-player-per-role.md) | v0.4 | Coordination UI when multiple people share a role |

See [`../roadmap.md`](../roadmap.md) for the full version plan and the
"intentionally not building" list.
